Loops

Introduction

Loops in JavaScript allow you to execute a block of code multiple times. They are useful for iterating over arrays, objects, and performing repetitive tasks efficiently.

Types of Loops in JavaScript

JavaScript provides several types of loops:

  • For Loop
  • While Loop
  • Do-While Loop
  • For-In Loop (used for objects)
  • For-Of Loop (used for arrays and iterable objects)

1. for Loop

Syntax

for (initialization; condition; increment/decrement) {
    // Code to execute
}

Example

for (let i = 1; i <= 5; i++) {
    console.log("Iteration " + i);
}

Explanation

  • Initialization: let i = 1; (Loop starts with i = 1)
  • Condition: i <= 5; (Loop runs while this is true)
  • Increment: i++ (Increases i by 1 in each iteration)

Output

Iteration 1
Iteration 2
Iteration 3
Iteration 4
Iteration 5

Loop with Array

let fruits = ["Apple", "Banana", "Mango"];
for (let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}

2. while Loop

Explanation

A while loop is used when the number of iterations is not known beforehand. It continues as long as the condition remains true.

Syntax

while (condition) {
    // Code to execute
}

Example

let count = 1;
while (count <= 5) {
    console.log("Count: " + count);
    count++;
}

Explanation

  • The loop checks count <= 5 before each iteration.
  • If the condition is true, the loop runs and increments count.

Output

Count: 1
Count: 2
Count: 3
Count: 4
Count: 5

3. do-while Loop

Explanation

A do-while loop is similar to a while loop, but it guarantees at least one execution, even if the condition is false.

Syntax

do {
    // Code to execute
} while (condition);

Example

let num = 1;
do {
    console.log("Number: " + num);
    num++;
} while (num <= 5);

Explanation

  • The code inside the do block runs once before checking the condition.
  • If the condition is true, it continues looping.

Output

Number: 1
Number: 2
Number: 3
Number: 4
Number: 5

4. for-in Loop (Used for Objects)

Syntax

for (key in object) {
    // Code to execute
}

Example

let student = { name: "John", age: 20, grade: "A" };

for (let key in student) {
    console.log(key + ": " + student[key]);
}

Output

name: John
age: 20
grade: A

5. for-of Loop (Used for Arrays and Iterables)

Syntax

for (variable of iterable) {
    // Code to execute
}

Example

let colors = ["Red", "Green", "Blue"];

for (let color of colors) {
    console.log(color);
}

Output

Red
Green
Blue

Loop Control Statements

1. break Statement

Used to exit a loop before it finishes.

for (let i = 1; i <= 5; i++) {
    if (i === 3) {
        break;
    }
    console.log(i);
}

Output

1
2

2. continue Statement

Skips the current iteration and moves to the next.

for (let i = 1; i <= 5; i++) {
    if (i === 3) {
        continue;
    }
    console.log(i);
}

Output

1
2
4
5

Comparison of Loops

Loop Type Best Used When
for When the number of iterations is known
while When the number of iterations is unknown
do-while When you need at least one execution
for-in When iterating over object properties
for-of When iterating over array elements

Conclusion

  • Loops are essential for performing repetitive tasks in JavaScript.
  • Use for loops when the number of iterations is known.
  • while and do...while loops are useful when the condition depends on dynamic input or logic.
  • for...in is ideal for iterating over object properties.
  • break and continue allow for more control over loop execution.

Assignment

  1. Write a for loop that prints numbers from 1 to 20, skipping multiples of 3 using continue.
  2. Create a while loop that prints even numbers from 1 to 10.
  3. Write a program that keeps asking the user for a number until they enter a number greater than 10 (Use do-while).

Assignment Answers

1. Write a for loop that prints numbers from 1 to 20, skipping multiples of 3 using continue.

for (let i = 1; i <= 20; i++) {
  if (i % 3 === 0) {
      continue;
  }
  console.log(i);
}

2. Create a while loop that prints even numbers from 1 to 10.

let num = 1;
while (num <= 10) {
  if (num % 2 === 0) {
      console.log(num);
  }
  num++;
}

3. Write a program that keeps asking the user for a number until they enter a number greater than 10 (Use do-while).

let number;
do {
    number = parseInt(prompt("Enter a number greater than 10:"));
} while (number <= 10);
