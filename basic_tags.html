Basic Tags

Heading Tags

Used to define headings, from <h1> (largest) to <h6> (smallest).

<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Sub-subheading</h3>

Note: Only one <h1> is recommended per page for SEO.

Paragraph Tag

<p> defines a paragraph.

<p>This is a paragraph of text.</p>

Horizontal Rule & Line Break

  • <hr>: Inserts a horizontal line (thematic break).
  • <br>: Inserts a line break without starting a new paragraph.
<p>Line one.<br>Line two.</p>
<hr>

Anchor Tag (<a>)

Creates hyperlinks.

<a href="https://www.example.com">Visit Example</a>
  • href is the destination URL.
  • Can link to external sites, internal pages, or sections within the same page using #id.

Image Tag

Displays an image.

<img src="image.jpg" alt="A description of the image">
  • src: Path to image.
  • alt: Text shown if image fails to load (important for accessibility).

Lists

Unordered List

<ul>
  <li>Item One</li>
  <li>Item Two</li>
</ul>

Ordered List

<ol>
  <li>First</li>
  <li>Second</li>
</ol>

Strong, Emphasis, and Comments

<strong>Important text</strong>
<em>Italicized text</em>
<!-- This is a comment -->
  • <strong>: Semantically important.
  • <em>: Emphasized text.
  • <!-- -->: Used to add comments in HTML.

Div and Span

  • <div>: Block-level container.
  • <span>: Inline container.
<div>This is a block container</div>
<span>This is inline</span>

Quiz

  1. Which tag is used for the largest heading?
    a) <head>
    b) <h6>
    c) <h1>
    d) <title>
  2. What does the <br> tag do?
    a) Starts a new paragraph
    b) Breaks the page
    c) Inserts a line break
    d) Adds bold text
  3. Which tag creates an unordered list?
    a) <ol>
    b) <li>
    c) <ul>
    d) <list>
  4. What is the purpose of the alt attribute in <img>?
    a) Change alignment
    b) Improve image quality
    c) Add text description
    d) Add title to the image
  5. The <div> tag is an inline container.
    a) True
    b) False
  6. The <strong> tag makes text bold and adds importance.
    a) True
    b) False

Answers:

  1. (c)
  2. (c)
  3. (c)
  4. (c)
  5. (b)
  6. (a)

Assignment

Create a simple webpage using basic HTML tags covered in this lecture.

You must write the HTML code for a personal introduction page that includes the following:

  • Use all six heading tags (<h1> to <h6>) at least once.
  • Add at least two paragraphs.
  • Use <hr> and <br> appropriately.
  • Create a hyperlink to any external website.
  • Display one image with alt text.
  • Include both ordered and unordered lists.
  • Add a <strong> and <em> in one of your paragraphs.
  • Wrap different sections in <div> tags.
  • Add a comment in your code.
  • Use a <span> to style or highlight a part of your text.
  • Make sure your code is properly indented and readable.

Bonus (Optional):

Try linking to another HTML file using <a href="anotherfile.html">
