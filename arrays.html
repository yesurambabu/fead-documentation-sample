Arrays

Introduction

Declaring and Initializing Arrays

There are multiple ways to declare an array in JavaScript:

Using Array Literal (Recommended)

let fruits = ["Apple", "Banana", "Mango", "Orange"];

Using the new Array() Constructor

let numbers = new Array(10, 20, 30, 40);

Creating an Empty Array and Adding Elements Later

let colors = [];
colors[0] = "Red";
colors[1] = "Blue";

Accessing Array Elements

let fruits = ["Apple", "Banana", "Mango"];
console.log(fruits[0]); // Output: Apple
console.log(fruits[1]); // Output: Banana
console.log(fruits[2]); // Output: Mango

Modifying Array Elements

let fruits = ["Apple", "Banana", "Mango"];
fruits[1] = "Grapes";
console.log(fruits); // Output: ["Apple", "Grapes", "Mango"]

Array Properties

let fruits = ["Apple", "Banana", "Mango"];
console.log(fruits.length);  // Output: 3

Array Methods

1. Adding and Removing Elements

MethodDescriptionExample
push()Adds an element to the endarr.push("New")
pop()Removes the last elementarr.pop()
unshift()Adds an element to the beginningarr.unshift("New")
shift()Removes the first elementarr.shift()
let fruits = ["Apple", "Banana", "Mango"];
fruits.push("Orange");
fruits.pop();
fruits.unshift("Grapes");
fruits.shift();

2. Merging and Slicing

MethodDescriptionExample
concat()Merges two arraysarr1.concat(arr2)
slice(start, end)Extracts a portion of an arrayarr.slice(1,3)
let arr1 = [1, 2, 3];
let arr2 = [4, 5, 6];
let merged = arr1.concat(arr2);
let sliced = merged.slice(2, 5);

3. Searching for Elements

MethodDescriptionExample
indexOf(value)Returns index of first occurrencearr.indexOf(3)
lastIndexOf(value)Returns last occurrence indexarr.lastIndexOf(3)
includes(value)Checks if value existsarr.includes(3)
let numbers = [10, 20, 30, 40, 50];
console.log(numbers.indexOf(30));
console.log(numbers.includes(25));

4. Sorting and Reversing

MethodDescriptionExample
sort()Sorts the arrayarr.sort()
reverse()Reverses order of elementsarr.reverse()
let fruits = ["Banana", "Apple", "Mango"];
fruits.sort();
fruits.reverse();

5. Joining and Splitting

MethodDescriptionExample
join(separator)Joins array into stringarr.join(", ")
split(separator)Splits string into array"text".split(" ")
let words = ["JavaScript", "is", "fun"];
let sentence = words.join(" ");
let newArray = sentence.split(" ");

Multidimensional Arrays

let matrix = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
];
console.log(matrix[1][2]);

Looping Through an Array

let numbers = [10, 20, 30, 40];
for (let i = 0; i < numbers.length; i++) {
  console.log(numbers[i]);
}
let fruits = ["Apple", "Banana", "Mango"];
fruits.forEach(function(fruit) {
  console.log(fruit);
});
let numbers = [1, 2, 3];
let squared = numbers.map(num => num * num);
console.log(squared);

Conclusion

  • Arrays store multiple values in a single variable.
  • They can be modified dynamically using methods.
  • Loops help iterate through arrays efficiently.
