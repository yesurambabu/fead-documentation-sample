JavaScript Form Validation

Form validation is the process of checking user input before it's sent to the server. JavaScript allows you to implement both client-side and custom validation rules using conditions, regular expressions, and DOM manipulation.

Why Validate Forms?

  • Ensure required fields are filled.
  • Prevent invalid data formats (like emails or phone numbers).
  • Improve user experience by providing instant feedback.
  • Reduce load on the server by avoiding unnecessary requests.

Types of Validation

  • Required Field Check: Ensure input is not empty.
  • Pattern Matching: Use regular expressions to check formats (e.g., email, phone number).
  • Length Checks: Minimum or maximum characters.
  • Numeric Checks: Ensure only numbers are entered.

Example: Validating a Simple Registration Form

<form id="registerForm">
  <input type="text" id="username" placeholder="Username"><br>
  <input type="text" id="email" placeholder="Email"><br>
  <input type="text" id="phone" placeholder="Phone Number"><br>
  <button type="submit">Submit</button>
</form>

<script>
document.getElementById("registerForm").addEventListener("submit", function(e) {
  e.preventDefault();

  const username = document.getElementById("username").value.trim();
  const email = document.getElementById("email").value.trim();
  const phone = document.getElementById("phone").value.trim();

  const userPattern = /^[a-zA-Z0-9]+$/;
  const emailPattern = /^[a-zA-Z0-9._]+@[a-z]+\.[a-z]{2,3}$/;
  const phonePattern = /^\d{10}$/;

  if (!userPattern.test(username)) {
    alert("Invalid username! Only alphanumeric characters are allowed.");
    return;
  }
  if (!emailPattern.test(email)) {
    alert("Invalid email format.");
    return;
  }
  if (!phonePattern.test(phone)) {
    alert("Phone number must be exactly 10 digits.");
    return;
  }

  alert("All inputs are valid!");
});
</script>

Assignment

Build a registration form with the following fields:

  • Full Name (Required, only letters and spaces)
  • Email (Must be a valid email format)
  • Password (Minimum 6 characters)
  • Mobile Number (Exactly 10 digits)

Use JavaScript to validate these fields before submission. Display relevant alerts if the validation fails.

Solution Example: Validating Registration Form

<form id="regForm">
  <input type="text" id="fullname" placeholder="Full Name"><br>
  <input type="email" id="email" placeholder="Email"><br>
  <input type="password" id="password" placeholder="Password"><br>
  <input type="text" id="mobile" placeholder="Mobile Number"><br>
  <button type="submit">Register</button>
</form>

<script>
document.getElementById("regForm").addEventListener("submit", function(e) {
  e.preventDefault();

  const fullname = document.getElementById("fullname").value.trim();
  const email = document.getElementById("email").value.trim();
  const password = document.getElementById("password").value.trim();
  const mobile = document.getElementById("mobile").value.trim();

  const namePattern = /^[a-zA-Z ]+$/;
  const emailPattern = /^[^\\s@]+@[^\\s@]+\\.[^\\s@]{2,}$/;
  const mobilePattern = /^\\d{10}$/;

  if (!namePattern.test(fullname)) {
    alert("Full name must contain only letters and spaces.");
    return;
  }
  if (!emailPattern.test(email)) {
    alert("Please enter a valid email.");
    return;
  }
  if (password.length < 6) {
    alert("Password must be at least 6 characters long.");
    return;
  }
  if (!mobilePattern.test(mobile)) {
    alert("Mobile number must be 10 digits.");
    return;
  }

  alert("Form submitted successfully!");
});
</script>
