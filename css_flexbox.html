CSS Flexbox

Introduction

The Flexible Box Layout, commonly known as Flexbox, is a one-dimensional layout model used to design flexible and responsive layout structures. It helps distribute space along a single column or row, aligning elements intelligently depending on the container's available space.

To use Flexbox, set the containerâ€™s display property to flex or inline-flex.

.container {
  display: flex;
}

flex-direction

Defines the direction in which the flex container places its flex items.

Property Name Value Description
flex-direction row Items are placed in a row from left to right (default).
row-reverse Items are placed in a row from right to left.
column Items are placed in a column from top to bottom.
column-reverse Items are placed in a column from bottom to top.

Example:

.container {
  display: flex;
  flex-direction: column-reverse;
}

flex-wrap

Specifies whether flex items should wrap or stay on a single line.

Property Name Value Description
flex-wrap nowrap All flex items will be on one line (default).
wrap Flex items will wrap onto multiple lines from top to bottom.
wrap-reverse Flex items will wrap onto multiple lines from bottom to top.

Example:

.container {
  display: flex;
  flex-wrap: wrap-reverse;
}

justify-content

Defines how flex items are aligned along the main axis.

Property Name Value Description
justify-content flex-start Items are packed toward the start of the main axis (default).
flex-end Items are packed toward the end of the main axis.
center Items are centered along the main axis.
space-between Items are evenly distributed with the first item at the start and last at the end.
space-around Items are evenly distributed with space around them.
space-evenly Items are evenly distributed with equal space between and around them.

Example:

.container {
  display: flex;
  justify-content: space-evenly;
}

align-items

Aligns flex items along the cross axis (perpendicular to the main axis).

Property Name Value Description
align-items stretch Items stretch to fill the container (default).
flex-start Items are aligned to the start of the cross axis.
flex-end Items are aligned to the end of the cross axis.
center Items are centered along the cross axis.
baseline Items are aligned such that their baselines align.

Example:

.container {
  display: flex;
  align-items: baseline;
}

align-content

Aligns multiple lines of content along the cross axis when there is extra space.

Property Name Value Description
align-content flex-start Lines are packed to the start of the container.
flex-end Lines are packed to the end of the container.
center Lines are centered in the container.
space-between Lines are evenly distributed; first at start, last at end.
space-around Lines are evenly distributed with space around each line.
stretch Lines stretch to fill the remaining space (default).

Example:

.container {
  display: flex;
  flex-wrap: wrap;
  align-content: space-around;
}

align-self

Overrides the align-items property for a specific flex item.

Property Name Value Description
align-self auto Inherits the value from align-items (default).
flex-start Aligns the item to the start of the cross axis.
flex-end Aligns the item to the end of the cross axis.
center Centers the item along the cross axis.
baseline Aligns the item with the baseline of others.
stretch Stretches the item to fill the container.

Example:

.item {
  align-self: stretch;
}

Quiz

  1. Which CSS property defines the direction in which flex items are placed?
    a) justify-content
    b) align-items
    c) flex-direction
    d) align-content
  2. What value of justify-content distributes items with equal space around them?
    a) space-between
    b) space-evenly
    c) space-around
    d) center
  3. Which property allows items to wrap into multiple lines?
    a) flex-flow
    b) align-content
    c) flex-wrap
    d) align-items
  4. What is the default value of align-items?
    a) center
    b) flex-start
    c) flex-end
    d) stretch
  5. Which property is used to override alignment for a specific flex item?
    a) align-content
    b) align-self
    c) align-items
    d) justify-content

Answers

  1. c) flex-direction
  2. c) space-around
  3. c) flex-wrap
  4. d) stretch
  5. b) align-self

Assignment

Create a flex container with three boxes. Use the following requirements:

  • Set the direction to column.
  • Allow the items to wrap if the screen is narrow.
  • Center all items both horizontally and vertically.
  • Use individual alignment for the second item to align it to the flex-start of the cross-axis.

Assignment Solution

<!DOCTYPE html>
<html>
<head>
  <style>
    .container {
      display: flex;
      flex-direction: column;
      flex-wrap: wrap;
      justify-content: center;
      align-items: center;
      height: 100vh;
    }
    .box {
      width: 100px;
      height: 100px;
      margin: 10px;
      background-color: lightblue;
    }
    .box:nth-child(2) {
      align-self: flex-start;
    }
  </style>
</head>
<body>
  <div class="container">
    <div class="box">Box 1</div>
    <div class="box">Box 2</div>
    <div class="box">Box 3</div>
  </div>
</body>
</html>
