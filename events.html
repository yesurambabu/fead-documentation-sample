Events

Introduction

Events are actions or occurrences that happen in the browser, often triggered by user interactions such as clicking a button, typing in a text field, or moving the mouse. JavaScript allows us to handle these events using event listeners.

What Are JavaScript Events?

JavaScript events are used to execute a function when an action occurs. Examples of events include:

  • Clicking a button (click event)
  • Typing in an input field (keyup, keydown, keypress events)
  • Moving the mouse (mousemove, mouseover, mouseout events)
  • Submitting a form (submit event)
  • Loading a webpage (load event)

Event Handling in JavaScript

There are three main ways to handle events in JavaScript:

  • Inline Event Handlers (using HTML attributes)
  • JavaScript Event Properties
  • Event Listeners

Inline Event Handlers (Using HTML Attributes)

The simplest way to handle events is by adding an event directly inside an HTML tag.

Example: Click Event in HTML

<button onclick="alert('Button Clicked!')">Click Me</button>

ðŸ”¹ This works but is not recommended for large-scale applications.

JavaScript Event Properties

You can assign event handler functions directly to HTML elements using JavaScript properties.

Example: Click Event Using JavaScript Property

<button id="btn">Click Me</button>

<script>
  let button = document.getElementById("btn");
  button.onclick = function () {
      alert("Button was clicked!");
  };
</script>

ðŸ”¹ The event handler (onclick) is assigned to the button element.

Event Listeners (addEventListener)

Using addEventListener() is the best way to handle events because:

  • It allows multiple event handlers for the same event.
  • It separates JavaScript from HTML for cleaner code.

Example: Using addEventListener()

<button id="myButton">Click Me</button>

<script>
  let btn = document.getElementById("myButton");
  btn.addEventListener("click", function () {
      alert("You clicked the button!");
  });
</script>

ðŸ”¹ This method is preferred in modern JavaScript.

Common JavaScript Events

Event TypeDescription

dblclickWhen an element is double-clicked

mouseoutWhen the mouse leaves an element

keyupWhen a key is released
keypressWhen a key is pressed down

Example: Handling Different Events

Mouse Events

<button id="hoverBtn">Hover Over Me</button>

<script>
  let hoverBtn = document.getElementById("hoverBtn");

  hoverBtn.addEventListener("mouseover", function () {
      hoverBtn.style.backgroundColor = "yellow";
  });

  hoverBtn.addEventListener("mouseout", function () {
      hoverBtn.style.backgroundColor = "";
  });
</script>

ðŸ”¹ Changes the button color when hovered.

Keyboard Events

<input type="text" id="textInput" placeholder="Type something...">
<p id="output"></p>

<script>
  let input = document.getElementById("textInput");
  let output = document.getElementById("output");

  input.addEventListener("keyup", function () {
      output.innerText = "You typed: " + input.value;
  });
</script>

ðŸ”¹ Displays what the user types in real-time.

Form Submission Event

<form id="myForm">
  <input type="text" id="name" placeholder="Enter your name">
  <button type="submit">Submit</button>
</form>

<script>
  let form = document.getElementById("myForm");

  form.addEventListener("submit", function (event) {
      event.preventDefault(); // Prevents page reload
      alert("Form Submitted!");
  });
</script>

ðŸ”¹ Prevents default form submission behavior.

Conclusion

  • JavaScript events help make web pages interactive.
  • onclick, onmouseover, onkeydown, and onsubmit are commonly used.
  • addEventListener() is the best practice for handling events.

Quiz Questions

  1. Which of the following is NOT a JavaScript event?
    a) click
    b) mouseover
    c) type
    d) keydown
  2. What is the best method to handle multiple event listeners?
    a) element.onclick
    b) addEventListener()
    c) event.handle
    d) event.call()
  3. What does event.preventDefault() do?
    a) Stops event propagation
    b) Prevents the default browser behavior
    c) Prevents JavaScript execution
    d) Reloads the page
  4. What is the correct way to remove an event listener?
    a) element.removeEventListener("click", functionName)
    b) element.off("click", functionName)
    c) element.deleteEvent("click")
    d) element.stopListening("click")
  5. Which event fires when a user presses a key and releases it?
    a) keydown
    b) keyup
    c) keypress
    d) release

Quiz Answers

  1. c) type
  2. b) addEventListener()
  3. b) Prevents the default browser behavior
  4. a) element.removeEventListener("click", functionName)
  5. b) keyup

Assignment Answers

1. Create a button that, when clicked, changes the background color of the page.

<button onclick="document.body.style.backgroundColor='lightblue'">Change Background</button>

2. Create an input field. When the user types, display the number of characters typed below the field.

<input type="text" id="inputText">
<p id="charCount"></p>

<script>
  let input = document.getElementById("inputText");
  let count = document.getElementById("charCount");

  input.addEventListener("input", function () {
      count.innerText = "Characters typed: " + input.value.length;
  });
</script>

3. Create an image that changes when hovered and reverts when mouse leaves.

<img id="hoverImage" src="image1.jpg" alt="Hover Image">

<script>
  let img = document.getElementById("hoverImage");

  img.addEventListener("mouseover", function () {
      img.src = "image2.jpg";
  });

  img.addEventListener("mouseout", function () {
      img.src = "image1.jpg";
  });
</script>

4. Create a form that shows an error if submitted empty.

<form id="myForm">
  <input type="text" id="name">
  <button type="submit">Submit</button>
  <p id="error" style="color:red;"></p>
</form>

<script>
  let form = document.getElementById("myForm");
  let nameInput = document.getElementById("name");
  let error = document.getElementById("error");

  form.addEventListener("submit", function (e) {
      e.preventDefault();
      if (nameInput.value.trim() === "") {
          error.textContent = "Name field cannot be empty!";
      } else {
          error.textContent = "";
          alert("Form submitted successfully!");
      }
  });
</script>
