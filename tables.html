Tables

Introduction

Tables in HTML are used to display data in a structured row and column format. They are commonly used for data representation, such as schedules, pricing tables, reports, and comparison charts.

Basic Structure of an HTML Table

A table in HTML is created using the <table> element, with rows (<tr>), columns (<td>), and headers (<th>).

Basic Example: Simple Table

<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
    <th>City</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>22</td>
    <td>New York</td>
  </tr>
  <tr>
    <td>Bob</td>
    <td>25</td>
    <td>Los Angeles</td>
  </tr>
</table>

<table>: Defines the table.

<tr>: Defines a table row.

<th>: Defines a table header (bold and centered by default).

<td>: Defines a table data cell.

Adding Borders, Spacing, and Padding

Bordered Table

<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
    <th>City</th>
  </tr>
  <tr>
    <td>John</td>
    <td>30</td>
    <td>Chicago</td>
  </tr>
</table>

The border="1" attribute adds borders to the table.

Styling with CSS (Spacing & Padding)

<style>
  table {
    border-collapse: collapse;
    width: 50%;
  }
  th, td {
    border: 1px solid black;
    padding: 10px;
    text-align: center;
  }
</style>
<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
    <th>City</th>
  </tr>
  <tr>
    <td>John</td>
    <td>30</td>
    <td>Chicago</td>
  </tr>
</table>

border-collapse: collapse; – Merges table borders.

padding: 10px; – Adds space inside cells.

text-align: center; – Aligns text.

Merging Cells (Rowspan & Colspan)

Merging Columns (colspan)

<table border="1">
  <tr>
    <th colspan="2">Full Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>John</td>
    <td>Doe</td>
    <td>28</td>
  </tr>
</table>

colspan="2" makes one cell span two columns.

Merging Rows (rowspan)

<table border="1">
  <tr>
    <th>Name</th>
    <th>City</th>
  </tr>
  <tr>
    <td rowspan="2">Alice</td>
    <td>New York</td>
  </tr>
  <tr>
    <td>Los Angeles</td>
  </tr>
</table>

rowspan="2" makes one cell span two rows.

Adding Table Headers and Footers

Table with <thead>, <tbody>, and <tfoot>

<table border="1">
  <thead>
    <tr>
        <th>Name</th>
        <th>Age</th>
        <th>City</th>
    </tr>
  </thead>
  <tbody>
    <tr>
        <td>David</td>
        <td>29</td>
        <td>Boston</td>
    </tr>
    <tr>
        <td>Emma</td>
        <td>24</td>
        <td>Seattle</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
        <td colspan="3">End of Data</td>
    </tr>
  </tfoot>
</table>

<thead> – Contains table headers.

<tbody> – Contains main data.

<tfoot> – Contains footer information.

Responsive Tables

Making a Table Scrollable

<style>
  .table-container {
    overflow-x: auto;
  }
</style>

<div class="table-container">
  <table border="1">
    <tr>
      <th>Name</th>
      <th>Age</th>
      <th>City</th>
    </tr>
    <tr>
      <td>Michael</td>
      <td>35</td>
      <td>San Francisco</td>
    </tr>
  </table>
</div>

The overflow-x: auto; makes the table scrollable on small screens.

Quiz

  1. Which HTML tag is used to create a table?
    a) <tb>
    b) <tr>
    c) <td>
    d) <table>
  2. What does the <th> tag represent?
    a) Table footer
    b) Table row
    c) Table header
    d) Table column
  3. What is the difference between colspan and rowspan?
    a) colspan merges rows, rowspan merges columns
    b) colspan merges columns, rowspan merges rows
    c) They are the same
    d) None of the above
  4. Which tag groups the main content of a table?
    a) <tbody>
    b) <thead>
    c) <tfoot>
    d) <th>
  5. How can you create a responsive table?
    a) Use <table border="1">
    b) Use <div style="overflow-x:auto;">
    c) Use <table style="float:right;">
    d) Use <table responsive>

Answers:

  1. (d)
  2. (c)
  3. (b)
  4. (a)
  5. (b)

Assignment

Create an interactive student database using an HTML table. The table should include:

  • Student Name
  • Roll Number
  • Course Name
  • Marks Obtained
  • Pass/Fail Status (Use CSS to highlight pass/fail in different colors)

Solution Example:

<style>
    table {
        width: 100%;
        border-collapse: collapse;
    }
    th, td {
        padding: 10px;
        border: 1px solid black;
        text-align: center;
    }
    .pass {
        background-color: lightgreen;
    }
    .fail {
        background-color: lightcoral;
    }
</style>

<table>
    <tr>
        <th>Student Name</th>
        <th>Roll Number</th>
        <th>Course</th>
        <th>Marks</th>
        <th>Status</th>
    </tr>
    <tr>
        <td>John Doe</td>
        <td>101</td>
        <td>Web Development</td>
        <td>85</td>
        <td class="pass">Pass</td>
    </tr>
    <tr>
        <td>Jane Smith</td>
        <td>102</td>
        <td>Web Development</td>
        <td>45</td>
        <td class="fail">Fail</td>
    </tr>
</table>
