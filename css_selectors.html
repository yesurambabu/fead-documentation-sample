CSS Selectors

CSS Basic Selectors

Element Selector

Targets all HTML elements of a specific type.

p {
    color: blue;
  }

Class Selector

Targets all elements with a specific class.

.highlight {
    background-color: yellow;
  }

ID Selector

Targets a specific element by its unique ID.

#header {
    font-size: 24px;
  }

Attribute Selector

Selects elements based on the presence or value of attributes.

input[type="text"] {
    border: 1px solid gray;
  }

Universal Selector

Targets all elements on the page.

* {
    margin: 0;
    padding: 0;
  }

Quiz Questions

  1. What does the .title selector target?
    a) An element with ID title
    b) All <title> tags
    c) All elements with class title
    d) The first <title> tag
  2. Which selector targets an element with the ID main?
    a) main
    b) #main
    c) .main
    d) *main
  3. What does the universal selector * do?
    a) Targets paragraphs
    b) Targets the body only
    c) Targets all elements
    d) Targets headings
  4. Choose the correct syntax for selecting an anchor tag with an href attribute
    a) a[href]
    b) a.here
    c) #a[href]
    d) href="a"
  5. What does input[type="text"] target?
    a) All input fields
    b) Input elements with type="text"
    c) Text elements
    d) Input with any type

Answers

  1. c) All elements with class title
  2. b) #main
  3. c) Targets all elements
  4. a) a[href]
  5. b) Input elements with type="text"

Assignment

Create an HTML page with the following:

  • A <h1> tag with an ID of main-heading
  • Two <p> tags, both using the class description
  • An <input> field with type="text"
  • A <div> with a class box
  • Use CSS to:
    • Style all paragraphs to have color: green
    • Make #main-heading blue
    • Give .box a border
    • Set padding for all elements using the universal selector
    • Style only the input with type="text" to have a light gray border

Assignment Solution

<h1 id="main-heading">Welcome</h1>
  <p class="description">This is the first paragraph.</p>
  <p class="description">This is the second paragraph.</p>
  <input type="text" />
  <div class="box">Box Content</div>
* {
    padding: 10px;
  }
  
  p {
    color: green;
  }
  
  #main-heading {
    color: blue;
  }
  
  .box {
    border: 1px solid black;
  }
  
  input[type="text"] {
    border: 1px solid lightgray;
  }
