CSS Animation

Introduction

CSS animations allow the transition of HTML elements from one style configuration to another. They are defined using keyframes and applied using the animation property.

Objective

Understand how to animate HTML elements using CSS properties like @keyframes, animation-name, animation-duration, animation-delay, and others.

Basic Syntax

@keyframes slideIn {
  from {
    transform: translateX(-100%);
  }
  to {
    transform: translateX(0);
  }
}

.box {
  animation-name: slideIn;
  animation-duration: 2s;
}

Common Animation Properties

Property Description
animation-name Name of the @keyframes to apply.
animation-duration Specifies how long the animation should take to complete one cycle.
animation-delay Delay before the animation starts.
animation-iteration-count Number of times the animation should repeat (or infinite).
animation-direction Direction of animation (normal, reverse, alternate, alternate-reverse).
animation-timing-function Speed curve of the animation (ease, linear, ease-in, etc.).
animation-fill-mode How styles are applied before/after animation (none, forwards, backwards, both).

Example

<div class="animated-box">Animate Me!</div>
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

.animated-box {
  width: 150px;
  height: 150px;
  background-color: tomato;
  color: white;
  display: flex;
  align-items: center;
  justify-content: center;
  animation-name: fadeIn;
  animation-duration: 2s;
}

Quiz

  1. Which property defines the name of the keyframes?
    a) animation-name
    b) animation-duration
    c) animation-fill-mode
    d) animation-delay
  2. Which value of animation-iteration-count makes the animation run forever?
    a) 0
    b) 1
    c) infinite
    d) forever
  3. What does animation-delay do?
    a) Speeds up the animation
    b) Delays animation start
    c) Reverses animation
    d) Removes animation
  4. Which property controls the direction in which the animation runs?
    a) animation-fill-mode
    b) animation-direction
    c) animation-timing-function
    d) animation-delay
  5. What is the purpose of @keyframes in CSS?
    a) Set up transitions
    b) Create animations
    c) Delay animation
    d) Group CSS properties

Quiz Answers

  1. a) animation-name
  2. c) infinite
  3. b) Delays animation start
  4. b) animation-direction
  5. b) Create animations

Assignment

Create a webpage with a div that animates using animation properties:

  • Define a @keyframes animation that changes background color from blue to green.
  • Apply this animation to a div with 200px width and 200px height.
  • Set duration to 3 seconds, delay of 1 second, and make it repeat 2 times.
  • Ensure the final color remains visible after animation completes.

Assignment Answer

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>CSS Animation Assignment</title>
  <style>
    @keyframes colorChange {
      from { background-color: blue; }
      to { background-color: green; }
    }

    .box {
      width: 200px;
      height: 200px;
      animation-name: colorChange;
      animation-duration: 3s;
      animation-delay: 1s;
      animation-iteration-count: 2;
      animation-fill-mode: forwards;
    }
  </style>
</head>
<body>
  <div class="box"></div>
</body>
</html>
