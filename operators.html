Operators

Introduction

Operators in JavaScript are symbols used to perform operations on variables and values. JavaScript supports various types of operators.

Types of JavaScript Operators

  1. Arithmetic Operators
  2. Assignment Operators
  3. Comparison Operators
  4. Logical Operators
  5. Bitwise Operators
  6. Ternary Operator

Arithmetic Operators

OperatorDescriptionExample
+Addition5 + 3 = 8
-Subtraction5 - 3 = 2
*Multiplication5 * 3 = 15
/Division10 / 2 = 5
%Modulus (Remainder)10 % 3 = 1
**Exponentiation2 ** 3 = 8

Example

let a = 10, b = 3;
console.log(a + b); // Output: 13
console.log(a % b); // Output: 1

Assignment Operators

OperatorExampleEquivalent To
=x = 5Assign 5 to x
+=x += 3x = x + 3
-=x -= 3x = x - 3
*=x *= 3x = x * 3
/=x /= 3x = x / 3

Example

let x = 10;
x += 5;
console.log(x); // Output: 15

Comparison Operators

OperatorDescriptionExample
==Equal to5 == "5" (true)
===Strict Equal to5 === "5" (false)
!=Not Equal5 != 3 (true)
!==Strict Not Equal5 !== "5" (true)
>Greater than5 > 3 (true)
<Less than5 < 3 (false)
>=Greater than or Equal5 >= 5 (true)
<=Less than or Equal5 <= 3 (false)

Example

console.log(10 > 5); // Output: true
console.log(10 === "10"); // Output: false

Logical Operators

OperatorDescriptionExample
&&AND(5 > 3 && 10 > 5) → true
||OR(5 > 3 || 10 < 5) → true
!NOT!(5 > 3) → false

Example

let a = true, b = false;
console.log(a && b); // Output: false
console.log(a || b); // Output: true

Bitwise Operators

OperatorDescriptionExample
&AND5 & 1 (1)
|OR5 | 1 (5)
^XOR5 ^ 1 (4)
<<Left Shift5 << 1 (10)
>>Right Shift5 >> 1 (2)

Ternary Operator

A shorthand for if-else statements:

let age = 18;
let status = (age >= 18) ? "Adult" : "Minor";
console.log(status); // Output: Adult

Assignment

Create an interactive web page that:

  • Performs arithmetic calculations based on user input.
  • Compares two values entered by the user.
  • Uses logical and ternary operators to determine conditions.

Assignment Solution

<!DOCTYPE html>
<html>
<head>
  <title>Operators Example</title>
  <script>
    function calculate() {
      let num1 = Number(document.getElementById("num1").value);
      let num2 = Number(document.getElementById("num2").value);
      let sum = num1 + num2;
      let comparison = (num1 > num2) ? "First is greater" : "Second is greater or equal";
      let isBothPositive = (num1 > 0 && num2 > 0);

      document.getElementById("output").innerHTML = 
        "Sum: " + sum + "<br>" +
        "Comparison: " + comparison + "<br>" +
        "Both positive? " + isBothPositive;
    }
  </script>
</head>
<body>
  <input id="num1" type="number">
  <input id="num2" type="number">
  <button onclick="calculate()">Check</button>
  <div id="output"></div>
</body>
</html>
