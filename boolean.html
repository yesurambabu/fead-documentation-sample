Boolean

Introduction

A Boolean is a data type in JavaScript that represents one of two values:

  • true
  • false

Booleans are commonly used in conditional statements, comparisons, and logical operations.

Declaring Boolean Variables

Boolean values can be assigned to variables directly:

let isStudent = true;
let hasLicense = false;

Booleans can also be created using the Boolean() function:

let value1 = Boolean(1);  // true
let value2 = Boolean(0);  // false
let value3 = Boolean("Hello");  // true
let value4 = Boolean("");  // false

Boolean Values in Conditions

let isRaining = true;

if (isRaining) {
    console.log("Take an umbrella!");
} else {
    console.log("Enjoy the sunshine!");
}

Falsy and Truthy Values

Falsy Values (evaluate to false in a Boolean context)

  • 0
  • "" (empty string)
  • null
  • undefined
  • NaN (Not a Number)
  • false

Example:

if (0) {
    console.log("This won't run because 0 is falsy.");
}

Truthy Values (evaluate to true in a Boolean context)

  • Any non-empty string ("Hello")
  • Any nonzero number (42, -1)
  • Any object ({}, [])
  • true

Example:

if ("Hello") {
    console.log("This will run because a non-empty string is truthy.");
}

Boolean Comparisons

Comparison operators return Boolean values.

Operator Description Example Output
== Equal to 5 == "5" true (loose comparison)
=== Strict equal to 5 === "5" false (strict comparison)
!= Not equal to 10 != 5 true
!== Strict not equal to 10 !== "10" true
> Greater than 10 > 5 true
< Less than 5 < 10 true
>= Greater than or equal to 10 >= 10 true
<= Less than or equal to 5 <= 10 true

Example:

let x = 10;
let y = 5;

console.log(x > y);  // true
console.log(x === "10");  // false (strict comparison)
console.log(x != y);  // true

Logical Operators

Logical operators are used to combine multiple Boolean expressions.

Operator Name Example Output
&& AND true && false false
|| OR true || false true
! NOT !true false

AND (&&) Operator

Returns true only if both conditions are true.

let hasID = true;
let hasTicket = true;

if (hasID && hasTicket) {
    console.log("You can enter the event.");
} else {
    console.log("Entry denied.");
}

OR (||) Operator

Returns true if at least one condition is true.

let isWeekend = false;
let isHoliday = true;

if (isWeekend || isHoliday) {
    console.log("You can relax!");
}

NOT (!) Operator

Reverses the Boolean value.

let isNight = false;
console.log(!isNight);  // true

Boolean Functions

Using Boolean in Functions

Booleans can be used in functions to return true or false.

function isEven(num) {
    return num % 2 === 0;
}

console.log(isEven(4));  // true
console.log(isEven(7));  // false

Boolean Conversion

You can convert any value to a Boolean using Boolean().

console.log(Boolean(100));  // true
console.log(Boolean(0));  // false
console.log(Boolean(""));  // false
console.log(Boolean("JavaScript"));  // true

Conclusion

  • JavaScript Booleans can hold true or false values.
  • Some values are truthy and others are falsy.
  • Boolean logic is essential in conditions, loops, and functions.

Assignment

  1. Create a voting eligibility checker that asks the user for their age.
  2. If the user is 18 or older, print "Eligible to vote"; otherwise, print "Not eligible".
  3. Use Boolean logic in your code.

Assignment Solution

let age = prompt("Enter your age:");
age = Number(age);

if (age >= 18) {
    console.log("Eligible to vote");
} else {
    console.log("Not eligible");
}
