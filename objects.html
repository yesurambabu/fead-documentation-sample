Built-in Objects

Introduction

JavaScript comes with several built-in objects that provide useful functionalities. These objects contain properties and methods that make it easier to manipulate data types, perform mathematical operations, handle dates, and more.

String Object

The String object is used to work with text.

Useful String Methods

Method Description Example
length Returns the length of the string. "Hello".length → 5
toUpperCase() Converts to uppercase. "hello".toUpperCase() → "HELLO"
toLowerCase() Converts to lowercase. "HELLO".toLowerCase() → "hello"
trim() Removes whitespace. " Hello ".trim() → "Hello"
charAt(index) Returns character at index. "Java".charAt(1) → "a"
includes(substr) Checks if string contains substring. "Hello".includes("He") → true
replace(a, b) Replaces text. "Hello".replace("H", "J") → "Jello"
split(delimiter) Splits string into an array. "A,B,C".split(",") → ["A", "B", "C"]

Example: String Manipulation

let text = " JavaScript ";
console.log(text.trim().toUpperCase()); // Outputs: "JAVASCRIPT"

Number Object

The Number object is used to work with numbers.

Useful Number Methods

MethodDescriptionExample
toFixed(n) Formats number with n decimals. (3.14159).toFixed(2) → "3.14"
parseInt() Parses a string and returns an integer. parseInt("42") → 42
parseFloat() Parses a string and returns a floating point number. parseFloat("3.14") → 3.14
isNaN() Checks if a value is NaN (Not a Number). isNaN("abc") → true
isFinite() Checks if value is a finite number. isFinite(123) → true
toString() Converts number to string. (123).toString() → "123"

Example: Number Conversion

let num = 3.14159;
console.log(num.toFixed(2)); // Outputs: "3.14"

Math Object

The Math object allows you to perform mathematical tasks.

Useful Math Methods

Method Description Example
Math.round() Rounds to the nearest integer. Math.round(4.6) → 5
Math.floor() Rounds down to the nearest integer. Math.floor(4.9) → 4
Math.ceil() Rounds up to the nearest integer. Math.ceil(4.1) → 5
Math.random() Returns a random number between 0 and 1. Math.random() → 0.123 (example)
Math.max() Returns the highest number. Math.max(1, 2, 3) → 3
Math.min() Returns the lowest number. Math.min(1, 2, 3) → 1
Math.sqrt() Returns the square root of a number. Math.sqrt(9) → 3

Example: Math Operations

let x = Math.ceil(4.3) + Math.floor(4.7);
console.log(x); // 9

Date Object

The Date object is used to work with dates and times.

Useful Date Methods

MethodDescriptionExample
new Date() Creates a new date object with the current date and time. new Date()
getFullYear() Returns the four-digit year. date.getFullYear()
getMonth() Returns the month (0–11). date.getMonth()
getDate() Returns the day of the month (1–31). date.getDate()
getDay() Returns the day of the week (0–6). date.getDay()
getHours() Returns the hour (0–23). date.getHours()
getMinutes() Returns the minutes (0–59). date.getMinutes()

Example: Date Manipulation

let now = new Date();
console.log(now.getFullYear()); // Outputs: current year
console.log(now.getMonth()); // Outputs: current month (0-11)

Array Object

The Array object is used to store multiple values in a single variable.

Useful Array Methods

Method Description Example
push() Adds elements to the end of an array. [1, 2].push(3) → [1, 2, 3]
pop() Removes the last element from an array. [1, 2, 3].pop() → [1, 2]
shift() Removes the first element of an array. [1, 2, 3].shift() → [2, 3]
unshift() Adds elements to the beginning of an array. [2, 3].unshift(1) → [1, 2, 3]
slice(start, end) Returns a shallow copy of a portion of an array. [1, 2, 3, 4].slice(1, 3) → [2, 3]
splice(start, deleteCount, items) Changes the contents of an array by removing or replacing elements. [1, 2, 3].splice(1, 1, 4) → [1, 4, 3]
includes() Checks if an array includes a certain value. [1, 2, 3].includes(2) → true
indexOf() Returns the first index of a value. [1, 2, 3].indexOf(2) → 1

Example: Array Manipulation

let arr = ["a", "b", "c"];
arr.push("d");
console.log(arr); // ["a", "b", "c", "d"]

RegExp Object

The RegExp object is used for matching text with a pattern. Regular expressions are patterns used to match character combinations in strings.

Useful RegExp Methods

MethodDescriptionExample
test() Tests for a match in a string. Returns true or false. /abc/.test("abc") → true
exec() Executes a search for a match and returns result array. /a/.exec("abc") → ["a"]
match() Returns matches using a pattern. "abc".match(/a/) → ["a"]
replace() Replaces matched substring with new substring. "abc".replace(/a/, "x") → "xbc"
search() Returns index of match or -1 if not found. "abc".search(/b/) → 1
split() Splits string based on pattern. "a,b,c".split(/,/) → ["a", "b", "c"]

Example: Regular Expression Matching

let str = "hello world";
  console.log(/world/.test(str)); // Outputs: true

Assignment

  • Create a program that displays today’s date in the format “DD/MM/YYYY”.
  • Create an array of five fruits and print them using a loop.
  • Write a function that converts a given string into uppercase using String methods.
  • Use Math.random() to simulate the rolling of a 6-sided dice.
  • Create a number and use toFixed() to show it with 2 decimal places.

Assignment Solution

1. Date Format:


let today = new Date();
let formatted = today.getDate() + "/" + (today.getMonth()+1) + "/" + today.getFullYear();
console.log(formatted);

2. Fruit Array:


let fruits = ["Apple", "Banana", "Orange", "Mango", "Pineapple"];
for (let fruit of fruits) {
  console.log(fruit);
}

3. Uppercase Function:


function toUpper(str) {
  return str.toUpperCase();
}
console.log(toUpper("hello"));

4. Dice Roll:


let dice = Math.floor(Math.random() * 6) + 1;
console.log("You rolled:", dice);

5. Number to Fixed:


let num = 4.56789;
console.log(num.toFixed(2));
