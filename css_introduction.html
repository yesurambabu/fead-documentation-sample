CSS Introduction

What is CSS?

CSS stands for Cascading Style Sheets. It is a language used to describe the style of an HTML document. CSS controls the layout, colors, fonts, spacing, and overall presentation of web content.

CSS allows developers to separate content (HTML) from design (CSS), improving maintainability and flexibility.

Why Use CSS?

  • Separates content from design
  • Reduces code duplication across pages
  • Improves website maintainability
  • Enables consistent look and feel
  • Reduces file sizes for faster loading

CSS Syntax

CSS rules are made up of a selector and a declaration block. A declaration contains property and value pairs.

selector {
  property: value;
}

Example:

p {
  color: red;
  font-size: 16px;
}

Types of CSS

Inline CSS

Styles added directly to the HTML element using the style attribute.

<p style="color: blue;">This is blue text.</p>

Internal CSS

Defined inside a <style> tag within the HTML <head>.

<head>
  <style>
    h1 {
      color: green;
    }
  </style>
</head>

External CSS

Linked using a separate .css file via the <link> tag.

<head>
  <link rel="stylesheet" href="styles.css">
</head>

CSS Selectors

Selectors are used to target HTML elements for styling.

Selector Description Example
Element Selects all elements of a type p {}
ID Selects a specific element with an ID #main {}
Class Selects elements with a given class .menu {}
Group Targets multiple elements h1, h2, p {}
Universal Selects all elements * {}

CSS Comments

Comments help annotate the CSS code and are ignored by browsers.

/* This is a CSS comment */

CSS Colors

Colors can be defined using names, hexadecimal values, or RGB.

color: red;
color: #ff0000;
color: rgb(255, 0, 0);

Cascading Order

When multiple rules apply to the same element, CSS follows this order:

  1. Inline styles (highest priority)
  2. Internal styles
  3. External styles
  4. Browser default (lowest priority)

Assignment

Objective

Create a simple webpage and apply CSS using all three types (inline, internal, and external).

Task Instructions

  • Create a webpage named css-intro.html.
  • Use a heading, a paragraph, and an image in the HTML.
  • Style the heading with inline CSS (color and font).
  • Add internal CSS to change paragraph color and size.
  • Link an external CSS file to style the image (width and border).

Submission Guidelines

  • Submit both css-intro.html and style.css.
  • Use semantic structure and indentation.

Assignment Solution

HTML File: css-intro.html

<!DOCTYPE html>
  <html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>CSS Introduction Assignment</title>
    <link rel="stylesheet" href="style.css">
    <style>
      p {
        color: #444;
        font-size: 18px;
        line-height: 1.5;
      }
    </style>
  </head>
  <body>
  
    <h1 style="color: darkblue; font-family: Arial, sans-serif;">Welcome to CSS Styling</h1>
  
    <p>This is a sample paragraph demonstrating internal CSS for text styling. 
    CSS makes websites beautiful and maintainable.</p>
  
    <img src="example.jpg" alt="Example Image" class="styled-img">
  
  </body>
  </html>

CSS File: style.css

.styled-img {
    width: 300px;
    border: 2px solid #888;
    border-radius: 5px;
  }

Notes

  • The heading uses inline CSS for color and font.
  • The paragraph is styled using internal CSS defined in a <style> block.
  • The image is styled using external CSS linked via style.css.
