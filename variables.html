Variables

Introduction

A variable in JavaScript is used to store data values. Variables act as containers for storing information that can be used and modified throughout the script.

Declaring Variables in JavaScript

JavaScript provides three ways to declare variables:

1. Using var (Old Method)

The var keyword was traditionally used to declare variables. However, it has some scope issues.

var message = "Hello, JavaScript!";
console.log(message);

2. Using let (Recommended for Modern JavaScript)

The let keyword allows block-scoped variables (limited to the block where they are defined).

let name = "John";
console.log(name);

3. Using const (For Constant Values)

The const keyword is used for variables that should not be reassigned.

const PI = 3.14;
console.log(PI);

Variable Naming Rules

  • Must start with a letter, underscore (_), or dollar sign ($).
  • Cannot start with a number.
  • Can contain letters, digits, underscores, and dollar signs.
  • Case-sensitive (age and Age are different variables).
  • Cannot be a reserved keyword (e.g., let, var, const).

Examples of Valid and Invalid Variables

let userName = "Alice"; // Valid
let _score = 100; // Valid
let $price = 50.99; // Valid
let 1number = 10; // Invalid (cannot start with a number)
let var = "test"; // Invalid (cannot use reserved keywords)

Variable Scope

1. Global Scope

A variable declared outside any function is global and can be accessed anywhere in the script.

var globalVar = "I'm global!";
function showGlobal() {
    console.log(globalVar);
}
showGlobal();

2. Local (Function) Scope

A variable declared inside a function is local and can only be accessed within that function.

function localExample() {
    let localVar = "I'm local!";
    console.log(localVar);
}
localExample();
// console.log(localVar); // Error: localVar is not defined

3. Block Scope (With let and const)

Variables declared inside a block {} using let or const are limited to that block.

if (true) {
    let blockVar = "I exist only inside this block";
    console.log(blockVar);
}
// console.log(blockVar); // Error: blockVar is not defined

Quiz: JavaScript Variables

  1. Which keyword should be used to declare a variable that never changes?
    a) var b) let
    c) const
    d) change
  2. What is the correct way to declare a JavaScript variable?
    a) variable x = 10;
    b) var x = 10;
    c) x := 10;
    d) let x == 10;
  3. Which of the following is a valid variable name?
    a) 2name
    b) _userAge
    c) var
    d) if
  4. What will happen if you try to reassign a const variable?
    a) It will change its value
    b) It will throw an error
    c) It will be ignored
    d) It will create a new variable
  5. What will the following code output?
    var x = 5;
if (true) {
    var x = 10;
}
console.log(x);
    a) 5
    b) 10
    c) Error
    d) Undefined

Answers

  1. c) const
  2. b) var x = 10;
  3. b) _userAge
  4. b) It will throw an error
  5. b) 10
