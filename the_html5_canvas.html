The HTML5 Canvas

What is HTML5 Canvas?

The <canvas> element is used to draw graphics via JavaScript. It acts as a container for graphics that can include shapes, text, images, and animations. Everything drawn inside the canvas must be handled through scripting.

Basic Canvas Syntax

<canvas id="myCanvas" width="400" height="300"></canvas>

This creates a canvas with a specified width and height. All actual drawing is done using JavaScript.

Getting the Context

const canvas = document.getElementById("myCanvas");
const ctx = canvas.getContext("2d");

The getContext("2d") method returns a drawing context for 2D rendering.

Drawing Shapes

Rectangle

ctx.fillStyle = "blue";
ctx.fillRect(50, 50, 100, 100);

Line

ctx.beginPath();
ctx.moveTo(10, 10);
ctx.lineTo(200, 10);
ctx.stroke();

Circle

ctx.beginPath();
ctx.arc(150, 75, 50, 0, 2 * Math.PI);
ctx.fillStyle = "red";
ctx.fill();

Drawing Text

ctx.font = "20px Arial";
ctx.fillStyle = "black";
ctx.fillText("Hello Canvas", 50, 200);

Drawing Images

const img = new Image();
img.onload = () => ctx.drawImage(img, 10, 10);
img.src = "image.jpg";

Canvas Use Cases

  • Games and animations
  • Data visualizations and charts
  • Interactive applications
  • Image editing

Assignment

Objective

Draw a scene using HTML5 canvas that includes:

  • A rectangle
  • A circle
  • A straight line
  • A short text label
  • An image

Submission Guidelines

  • Create an HTML file with a <canvas> element.
  • Use JavaScript to draw all items.
  • Include comments to explain your code.

Assignment Solution

<!DOCTYPE html>
<html>
<head>
  <title>Canvas Assignment</title>
  <style>
    canvas { border: 1px solid #ccc; }
  </style>
</head>
<body>
  <canvas id="myCanvas" width="500" height="400"></canvas>
  <script>
    const canvas = document.getElementById('myCanvas');
    const ctx = canvas.getContext('2d');

    // Rectangle
    ctx.fillStyle = 'skyblue';
    ctx.fillRect(20, 20, 100, 80);

    // Circle
    ctx.beginPath();
    ctx.arc(200, 60, 40, 0, 2 * Math.PI);
    ctx.fillStyle = 'green';
    ctx.fill();

    // Line
    ctx.beginPath();
    ctx.moveTo(10, 200);
    ctx.lineTo(300, 200);
    ctx.stroke();

    // Text
    ctx.font = '18px sans-serif';
    ctx.fillStyle = 'black';
    ctx.fillText('My Canvas Drawing', 150, 250);

    // Image
    const img = new Image();
    img.onload = function() {
      ctx.drawImage(img, 350, 50, 100, 100);
    };
    img.src = 'https://via.placeholder.com/100';
  </script>
</body>
</html>
