CSS Combiners

Introduction

CSS combiners are used to define relationships between selectors. They allow you to target elements based on their location relative to other elements in the HTML structure.

Grouping Selector (,)

Used to apply the same style to multiple, unrelated selectors.

h1, p, .note {
  color: blue;
}

Descendant Selector (space)

Targets elements that are nested inside a specified ancestor, at any depth.

div p {
  font-size: 16px;
}

Child Selector (>)

Targets only direct child elements.

ul > li {
  list-style: square;
}

Adjacent Sibling Selector (+)

Selects the first sibling element that comes immediately after a specific element.

h2 + p {
  color: green;
}

General Sibling Selector (~)

Selects all sibling elements that come after a specified element.

h2 ~ p {
  font-style: italic;
}

Chaining Selectors

Combines multiple selectors without space to target elements that match all conditions.

p.note.important {
  background-color: yellow;
}

Combining Combiners

You can use multiple combiners together to create very specific rules.

div > ul li a {
  text-decoration: none;
}

Quiz

  1. What does the selector div p select?
    a) Only direct children p of div
    b) All p elements that are children of div
    c) All p elements inside div at any level
    d) p elements after div
  2. What is the function of the > selector in CSS?
    a) Targets sibling elements
    b) Targets all nested elements
    c) Targets direct child elements only
    d) Targets adjacent elements
  3. Which selector applies the same style to multiple selectors?
    a) Descendant
    b) Chaining
    c) Grouping
    d) Sibling
  4. What does h2 + p select?
    a) All p tags in h2
    b) Any p sibling of h2
    c) The p element immediately following an h2
    d) First child p of h2
  5. What does this rule target: div > ul li a?
    a) All a tags
    b) All a tags inside li inside any ul
    c) a tags inside li that are inside a ul which is a direct child of div
    d) a tags that follow a div

Quiz Answers

  1. c) All p elements inside div at any level
  2. c) Targets direct child elements only
  3. c) Grouping
  4. c) The p element immediately following an h2
  5. c) a tags inside li that are inside a ul which is a direct child of div

Assignment

Objective

Create an HTML file that demonstrates the use of all the following combiners:

  • Grouping selector
  • Descendant selector
  • Child selector
  • Adjacent sibling selector
  • General sibling selector
  • Chained selector
  • Combination of combiners

Submission Guidelines

  • Use internal CSS.
  • Use clear text and borders/colors to show visual effects of the selectors.
  • Ensure your HTML is semantic and well-formatted.

Assignment Solution

<!DOCTYPE html>
<html>
<head>
  <style>
    h1, h2 {
      color: blue; /* Grouping */
    }

    div p {
      font-size: 16px; /* Descendant */
    }

    ul > li {
      color: green; /* Child */
    }

    h2 + p {
      background: yellow; /* Adjacent sibling */
    }

    h2 ~ p {
      font-style: italic; /* General sibling */
    }

    p.note.important {
      border: 1px solid red; /* Chaining */
    }

    section > div p span {
      color: purple; /* Combining combiners */
    }
  </style>
</head>
<body>

  <h1>Grouped Heading 1</h1>
  <h2>Grouped Heading 2</h2>

  <div>
    <p>Paragraph inside div (descendant)</p>
  </div>

  <ul>
    <li>Item 1</li>
    <li>Item 2</li>
  </ul>

  <h2>Heading 2</h2>
  <p>Paragraph next to h2 (adjacent)</p>
  <p>Another paragraph (general sibling)</p>

  <p class="note important">Chained Selector Example</p>

  <section>
    <div>
      <p><span>Nested Span for Combination</span></p>
    </div>
  </section>

</body>
</html>
