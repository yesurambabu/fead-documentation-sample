CSS Display

display: block

Makes the element start on a new line and take up the full width available. Common block elements include <div>, <p>, and headings like <h1> to <h6>.

div {
  display: block;
}

display: inline

Makes the element not start on a new line and only take up as much width as needed. Cannot set width, height, margin-top/bottom or padding-top/bottom.

span {
  display: inline;
}

display: inline-block

Displays the element in-line like inline but allows setting height and width like block.

button {
  display: inline-block;
  width: 100px;
  height: 30px;
}

display: none

Completely removes the element from the document flow; it will not be visible and will not occupy any space.

#hide-me {
  display: none;
}

Quiz Questions

  1. What does display: block; do?
    a) Makes the element appear in the same line
    b) Hides the element
    c) Makes the element take full width and start on a new line
    d) Makes the element disappear but still take space
  2. Which display value allows setting width and height but still behaves like inline?
    a) block
    b) inline-block
    c) inline
    d) none
  3. What happens when display: none; is applied?
    a) The element is hidden and takes up space
    b) The element becomes visible
    c) The element is hidden and does not take up space
    d) The element becomes a block
  4. Can you apply margin and padding to an element with display: inline;?
    a) Yes, fully
    b) Only left and right
    c) Only top and bottom
    d) No, not at all
  5. What type of element is <div> by default?
    a) Inline
    b) None
    c) Inline-block
    d) Block

Answers

  1. c) Makes the element take full width and start on a new line
  2. b) inline-block
  3. c) The element is hidden and does not take up space
  4. b) Only left and right
  5. d) Block

Assignment

Create an HTML page that demonstrates the different display values:

  • A <div> styled with display: block
  • A <span> styled with display: inline
  • A <button> styled with display: inline-block
  • A <p> styled with display: none
  • Use background colors and borders to clearly show how each display type behaves.

Assignment Solution

<div class="box1">Block Element</div>
<span class="box2">Inline Element</span>
<button class="box3">Inline-Block</button>
<p class="box4">This paragraph is hidden</p>

.box1 {
  display: block;
  background-color: lightblue;
  border: 1px solid blue;
}

.box2 {
  display: inline;
  background-color: lightgreen;
  border: 1px solid green;
}

.box3 {
  display: inline-block;
  background-color: lightcoral;
  border: 1px solid red;
  width: 150px;
  height: 40px;
}

.box4 {
  display: none;
}
