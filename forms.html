Forms

Introduction

HTML forms are used to collect user input and send it to a server for processing. Forms are essential in web applications for tasks like user registration, login, search, feedback, and payment processing.

Basic Structure of a Form

A form in HTML is created using the <form> element, which contains input fields, buttons, and other interactive elements.

<form action="submit.php" method="post">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    <input type="submit" value="Submit">
</form>

Key Elements in a Form

  • <form>: Defines a form.
  • action: Specifies the URL where form data is sent.
  • method: Defines how data is sent (GET or POST).
  • <label>: Defines a label for an input field.
  • <input>: Allows user input.
  • <submit>: A button to submit the form.

Form Attributes

Common Form Attributes

AttributeDescription
actionSpecifies where to send form data
methodDefines GET or POST request type
target_self, _blank, _parent, _top
autocompleteEnables or disables autofill (on/off)
novalidateDisables form validation

GET vs. POST Method

MethodDescriptionWhen to Use
GETSends data in the URLWhen data is not sensitive (e.g., search forms)
POSTSends data in the request bodyWhen data is sensitive (e.g., login, registration)

Input Fields in Forms

Text Input Field

<label for="username">Username:</label>
<input type="text" id="username" name="username">

Password Field

<label for="password">Password:</label>
<input type="password" id="password" name="password">

Email Field

<label for="email">Email:</label>
<input type="email" id="email" name="email">

Radio Buttons

<p>Gender:</p>
<input type="radio" id="male" name="gender" value="male">
<label for="male">Male</label>
<input type="radio" id="female" name="gender" value="female">
<label for="female">Female</label>

Checkboxes

<p>Select your hobbies:</p>
<input type="checkbox" id="reading" name="hobby" value="Reading">
<label for="reading">Reading</label>
<input type="checkbox" id="sports" name="hobby" value="Sports">
<label for="sports">Sports</label>

Dropdown List (Select Box)

<label for="country">Choose a country:</label>
<select id="country" name="country">
    <option value="USA">USA</option>
    <option value="UK">UK</option>
    <option value="India">India</option>
</select>

Textarea (Multi-line Input)

<label for="comments">Your comments:</label>
<textarea id="comments" name="comments" rows="4" cols="30"></textarea>

Date Picker

<label for="dob">Date of Birth:</label>
<input type="date" id="dob" name="dob">

Form Buttons

Submit Button

<input type="submit" value="Submit">

Reset Button

<input type="reset" value="Reset">

Button Element (Custom Actions)

<button type="button" onclick="alert('Button Clicked!')">Click Me</button>

Form Validation

Required Fields

<input type="text" name="name" required>

Minimum and Maximum Length

<input type="text" name="username" minlength="3" maxlength="10">

Pattern Matching

<input type="text" name="zipcode" pattern="[0-9]{5}" title="Enter a 5-digit ZIP code">
