Links and Navigation

1. Introduction

What are Links?

Links (or hyperlinks) connect different web pages or sections within the same page.

Links allow users to navigate across different resources on the web.

What is Navigation?

Navigation refers to the structured menu system on a website that helps users move between pages easily.

Good navigation improves user experience (UX) and SEO ranking.

2. Types of Links

  • External Links – Links to other websites.
  • Internal Links – Links within the same website.
  • Anchor Links – Links to a specific section of the same page.
  • Email and Telephone Links – Links that trigger an email or phone call.

3. Basic Structure of Links in HTML

Basic Hyperlink Structure

<a href="https://www.google.com">Visit Google</a>
  • The <a> (anchor) tag creates a link.
  • The href attribute specifies the link destination.
  • Clicking the text inside <a> will take the user to the specified URL.

4. Types of Links and Their Implementation

1. Internal Links

<a href="about.html">About Us</a>

2. External Links

<a href="https://www.wikipedia.org" target="_blank">Visit Wikipedia</a>

3. Anchor Links

<a href="#section2">Go to Section 2</a>

<h2 id="section2">Section 2</h2>
<p>This is Section 2 of the page.</p>

4. Email and Phone Links

<a href="mailto:someone@example.com">Email Us</a>
<a href="tel:+1234567890">Call Us</a>

5. Creating a Navigation Menu

Basic Navigation Bar Using HTML

<nav>
  <ul>
    <li><a href="index.html">Home</a></li>
    <li><a href="about.html">About</a></li>
    <li><a href="contact.html">Contact</a></li>
  </ul>
</nav>

Quiz

  1. Which HTML tag is used to create a hyperlink?
    a) <link>
    b) <a>
    c) <nav>
    d) <href>
  2. What does target="_blank" do in a link?
    a) Opens the link in the same tab
    b) Opens the link in a new tab
    c) Changes the text color
    d) Makes the link clickable
  3. How do you create a link to an email address?
    a) <a href="emailto:someone@example.com">Email</a>
    b) <a href="mailto:someone@example.com">Email</a>
    c) <a href="mail:someone@example.com">Email</a>
    d) <email>someone@example.com</email>
  4. Which tag is used to create a navigation menu?
    a) <nav>
    b) <menu>
    c) <ul>
    d) <navmenu>
  5. What is the purpose of anchor links (#id)?
    a) To open a new website
    b) To create a new navigation bar
    c) To navigate to a specific section within the same page
    d) To display an image

Answers:

  1. (b)
  2. (b)
  3. (b)
  4. (a)
  5. (c)

Assignment

Create a responsive navigation menu that includes:

  • A homepage with a structured navigation bar
  • Dropdown menus for categories
  • Anchor links to sections within a page
  • External links to social media pages

Solution Example: A Portfolio Navigation Bar

<!DOCTYPE html>
<html>
<head>
  <title>Portfolio</title>
  <style>
    nav ul {
      list-style-type: none;
      display: flex;
      background-color: #222;
      padding: 10px;
    }
    nav li {
      padding: 10px;
    }
    nav a {
      color: white;
      text-decoration: none;
    }
  </style>
</head>
<body>
  <nav>
    <ul>
      <li><a href="index.html">Home</a></li>
      <li><a href="about.html">About</a></li>
      <li><a href="portfolio.html">Portfolio</a></li>
      <li><a href="contact.html">Contact</a></li>
    </ul>
  </nav>
</body>
</html>
