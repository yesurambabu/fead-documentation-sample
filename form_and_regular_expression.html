Form and Regular Expression

HTML Forms

Introduction

An HTML <form> is used to gather user input. It can contain form controls like input fields, checkboxes, radio buttons, text areas, and submit buttons. Each input element collects specific types of data.

<form action="/submit" method="post">
  <label>Name: </label>
  <input type="text" name="username" required><br>
  <label>Email: </label>
  <input type="email" name="email" required><br>
  <input type="submit" value="Submit">
</form>

Regular Expressions

What is a Regular Expression?

A Regular Expression (RegEx) is a sequence of characters that form a search pattern. It is primarily used for string pattern matching and data validation. RegEx can be simple or very complex depending on the pattern you want to define.

Basic Syntax

Using the Literal notation:

const pattern = /expression/;

Or using the RegExp constructor:

const pattern = new RegExp("expression");

Regular Expression Tokens

Character Sets

Pattern Description Example
[abc] Match any one of a, b, or c /[abc]/ matches "a" in "cat"
[^abc] Match any character except a, b, or c /[^abc]/ matches "d" in "dog"
[a-z] Match any lowercase letter /[a-z]/ matches "h" in "hello"
[A-Z] Match any uppercase letter /[A-Z]/ matches "H" in "Hello"
[0-9] Match any digit /[0-9]/ matches "2" in "a2b"

Quantifiers

Symbol Description Example
* 0 or more times /a*/ matches "", "a", "aaa"
+ 1 or more times /a+/ matches "a", "aa"
? 0 or 1 time /a?/ matches "", "a"
{n} Exactly n times /a{3}/ matches "aaa"
{n,} n or more times /a{2,}/ matches "aa", "aaa", "aaaa"
{n,m} Between n and m times /a{2,4}/ matches "aa", "aaa", "aaaa"

Anchors and Special Characters

Symbol Description Example
^ Start of string /^a/ matches "a" at start of "apple"
$ End of string /e$/ matches "e" at end of "apple"
\d Any digit (0–9) /\d/ matches "5" in "abc5"
\D Any non-digit /\D/ matches "a" in "a1"
\w Word character (a-z, A-Z, 0-9, _) /\w/ matches "a", "Z", "1", "_"
\W Non-word character /\W/ matches "!" in "Hi!"
\s Whitespace character /\s/ matches space in "a b"
\S Non-whitespace /\S/ matches "a" in " a"
. Any character except newline /./ matches "a", "1", "!"
\ Escape character /\./ matches "." literally

Grouping and Alternation

Symbol Description Example
(abc) Group expression /(abc)+/ matches "abcabc"
a|b Either a or b /cat|dog/ matches "cat" or "dog"

Methods

Regular Expression Methods

Method Description Example
test() Returns true or false if match is found /a/.test("apple") → true
exec() Returns match result array or null /\d+/.exec("abc123") → ["123"]

String Methods Using RegExp

Method Description Example
match() Retrieves matches "abc123".match(/\d+/) → ["123"]
replace() Replaces matched substring "abc123".replace(/\d/, "#") → "abc#23"
search() Returns index of first match "abc123".search(/\d/) → 3
split() Splits string using regex "one,two,three".split(/,/) → ["one", "two", "three"]

Example:

const text = "My number is 9876543210";
const result = text.replace(/\d{10}/, "XXXXXXXXXX");
console.log(result); // "My number is XXXXXXXXXX"

const found = /\d+/.exec(text);
console.log(found[0]); // "9876543210"

console.log(text.match(/\d+/)); // ["9876543210"]

Validation Example: Email

const emailPattern = /^[a-zA-Z0-9._]+@[a-z]+\.[a-z]{2,3}$/;
emailPattern.test("user1@gmail.com"); // true
emailPattern.test("user@site");       // false

Assignment

Create a form with the following fields:

  • Username (must only contain letters and numbers)
  • Email (must be a valid email format)
  • Phone Number (must be 10 digits)

Use JavaScript with Regular Expressions to validate the input when the form is submitted. Show alerts if input is invalid.

Assignment Solution

<form id="userForm">
  <input type="text" id="username" placeholder="Username"><br>
  <input type="text" id="email" placeholder="Email"><br>
  <input type="text" id="phone" placeholder="Phone Number"><br>
  <button type="submit">Submit</button>
</form>

<script>
document.getElementById("userForm").addEventListener("submit", function(e) {
  e.preventDefault();

  const username = document.getElementById("username").value;
  const email = document.getElementById("email").value;
  const phone = document.getElementById("phone").value;

  const userPattern = /^[a-zA-Z0-9]+$/;
  const emailPattern = /^[a-zA-Z0-9._]+@[a-z]+\.[a-z]{2,3}$/;
  const phonePattern = /^\d{10}$/;

  if (!userPattern.test(username)) {
    alert("Invalid username!");
    return;
  }
  if (!emailPattern.test(email)) {
    alert("Invalid email!");
    return;
  }
  if (!phonePattern.test(phone)) {
    alert("Phone must be 10 digits!");
    return;
  }

  alert("All inputs are valid!");
});
</script>
