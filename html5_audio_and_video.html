HTML5 Audio and Video

Introduction to HTML5 Media Elements

HTML5 introduced native support for audio and video playback using the <audio> and <video> elements. These elements allow media embedding without plugins and support multiple file formats and user controls.

The <audio> Element

The <audio> tag embeds sound content in a document. It supports multiple file types for wider browser compatibility.

<audio controls>
  <source src="sound.mp3" type="audio/mpeg">
  <source src="sound.ogg" type="audio/ogg">
  Your browser does not support the audio element.
</audio>

Common Attributes

  • controls – shows playback controls
  • autoplay – plays automatically
  • loop – plays repeatedly
  • muted – starts muted

The <video> Element

The <video> element is used to embed video content. Like <audio>, it supports multiple sources and formats.

<video width="400" controls>
  <source src="video.mp4" type="video/mp4">
  <source src="video.ogg" type="video/ogg">
  Your browser does not support the video tag.
</video>

Common Attributes

  • width, height – defines dimensions
  • controls – displays UI controls
  • autoplay, loop, muted
  • poster – shows an image before video plays

Common Media Formats

Media Type Common Formats MIME Types
Audio MP3, Ogg, WAV audio/mpeg, audio/ogg
Video MP4, WebM, Ogg video/mp4, video/webm

Accessibility Tip

Always provide fallback text inside <audio> and <video> for unsupported browsers.

Your browser does not support the audio/video element.

Use the <track> element with <video> to include subtitles.

Assignment Solution

<!DOCTYPE html>
<html>
<head>
  <title>Media Showcase</title>
</head>
<body>

  <h2>Audio Sample</h2>
  <p>This is a demo audio file:</p>
  <audio controls>
    <source src="sample.mp3" type="audio/mpeg">
    <source src="sample.ogg" type="audio/ogg">
    Your browser does not support the audio element.
  </audio>

  <h2>Video Sample</h2>
  <p>This is a demo video with poster image and fallback:</p>
  <video width="400" controls poster="poster.jpg">
    <source src="sample.mp4" type="video/mp4">
    <source src="sample.webm" type="video/webm">
    Your browser does not support the video tag.
  </video>

</body>
</html>
