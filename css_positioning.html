CSS Positioning

1. position: static

This is the default positioning of all HTML elements.

Elements are placed according to the normal document flow.

top, right, bottom, and left have no effect.

Example

div {
  position: static;
}

2. position: relative

The element is positioned relative to its normal position.

Using top, right, bottom, or left will move it from where it would normally appear.

Example

div {
  position: relative;
  top: 10px;
  left: 20px;
}

3. position: absolute

The element is positioned relative to the nearest positioned ancestor (i.e., an ancestor with any position other than static).

If no such ancestor exists, it is positioned relative to the <html> element.

It is removed from the document flow.

Example

div {
  position: absolute;
  top: 50px;
  left: 100px;
}

4. position: fixed

The element is positioned relative to the browser window.

It does not move when the page is scrolled.

Commonly used for navigation bars, "back to top" buttons, etc.

Example

div {
  position: fixed;
  top: 0;
  right: 0;
}

5. position: sticky

The element behaves like relative until it reaches a specific scroll position, then it "sticks" like fixed.

Only works when a top, left, right, or bottom value is given.

Often used in headers and table of contents.

Example

div {
  position: sticky;
  top: 0;
}

Quiz Questions

  1. What is the default value of the position property?
    a) relative
    b) absolute
    c) static
    d) fixed
  2. Which position removes the element from the normal document flow and positions it relative to its nearest positioned ancestor?
    a) relative
    b) absolute
    c) fixed
    d) static
  3. Which position type will keep an element fixed at the top even when the user scrolls?
    a) absolute
    b) relative
    c) fixed
    d) sticky
  4. When does position: sticky; start acting like fixed?
    a) When the page loads
    b) When the element is clicked
    c) When the scroll position reaches its threshold
    d) It always behaves like fixed
  5. Which position respects top/right/bottom/left but keeps the element in document flow?
    a) static
    b) relative
    c) fixed
    d) absolute

Quiz Answers

  • Q1: c) static
  • Q2: b) absolute
  • Q3: c) fixed
  • Q4: c) When the scroll position reaches its threshold
  • Q5: b) relative

Assignment

Create a webpage with 5 <div> elements, each demonstrating a different position value:

  • One with position: static
  • One with position: relative and offset from the top-left
  • One with position: absolute inside a relatively positioned container
  • One with position: fixed at the bottom-right corner
  • One with position: sticky that sticks to the top on scroll

Add borders or background colors to clearly see the behavior of each element.

Assignment Solution

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>CSS Positioning Example</title>
  <style>
    body {
      margin: 0;
      font-family: Arial, sans-serif;
    }

    header {
      position: fixed;
      top: 0;
      width: 100%;
      height: 60px;
      background-color: #333;
      color: white;
      display: flex;
      align-items: center;
      justify-content: center;
      z-index: 1000;
    }

    .container {
      display: flex;
      margin-top: 70px; /* Space for fixed header */
      padding: 20px;
      position: relative;
    }

    .sidebar {
      width: 200px;
      height: 300px;
      background-color: #f4f4f4;
      position: sticky;
      top: 70px;
      padding: 10px;
    }

    main {
      flex: 1;
      padding: 20px;
      background-color: #e9e9e9;
      min-height: 500px;
      position: relative;
    }

    .relative-box {
      width: 200px;
      height: 100px;
      background-color: lightblue;
      position: relative;
      top: 20px;
      left: 30px;
      margin-bottom: 40px;
    }

    .absolute-box {
      width: 200px;
      height: 100px;
      background-color: salmon;
      position: absolute;
      top: 150px;
      left: 300px;
    }
  </style>
</head>
<body>

  <header>Fixed Header</header>

  <div class="container">
    <nav class="sidebar">Sticky Sidebar</nav>
    <main>
      <div class="relative-box">Relative Positioned Box</div>
      <div class="absolute-box">Absolute Positioned Box</div>
    </main>
  </div>

</body>
</html>
