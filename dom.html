DOM

Introduction

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, allowing JavaScript to manipulate HTML elements dynamically.

Understanding the DOM Tree

When a web page loads, the browser creates a DOM tree based on the HTML document. The DOM consists of:

  • Document: The root node of the tree.
  • Elements: HTML elements like <body>, <div>, <p>.
  • Attributes: Properties of elements (e.g., id, class).
  • Text Nodes: The text inside an HTML element.

Example HTML:

<!DOCTYPE html>
<html>
<head>
    <title>DOM Example</title>
</head>
<body>
    <h1 id="title">Hello, DOM!</h1>
    <p class="description">This is a paragraph.</p>
</body>
</html>

Corresponding DOM Tree:

Document
  ├── html
      ├── head
      │   └── title
      └── body
          ├── h1 (id="title")
          └── p (class="description")

Accessing DOM Elements

JavaScript provides methods to access and manipulate DOM elements:

1. Using getElementById()

let title = document.getElementById("title");
title.style.color = "blue"; // Changes text color

2. Using getElementsByClassName()

let paragraphs = document.getElementsByClassName("description");
paragraphs[0].style.fontSize = "20px";

3. Using getElementsByTagName()

let allParagraphs = document.getElementsByTagName("p");
console.log(allParagraphs.length); // Logs number of <p> elements

4. Using querySelector() and querySelectorAll()

let title = document.querySelector("#title"); // Selects by ID
let paragraphs = document.querySelectorAll(".description"); 
// Selects all elements with class "description"

Modifying DOM Elements

document.getElementById("title").innerHTML = "New Heading";
document.getElementById("title").style.backgroundColor = "yellow";

Adding and Removing Elements

1. Creating Elements

let newPara = document.createElement("p");
newPara.textContent = "This is a new paragraph.";
document.body.appendChild(newPara);

2. Removing Elements

let removeTitle = document.getElementById("title");
removeTitle.remove();

Quiz Questions

  1. What does DOM stand for?
    a) Document Object Model
    b) Data Object Management
    c) Document Order Module
    d) Dynamic Object Manipulation
  2. Which method is used to select an element by its ID?
    a) getElementsByClassName()
    b) querySelectorAll()
    c) getElementById()
    d) getElementsByTagName()
  3. What does document.createElement("div") do?
    a) Selects an existing <div> element
    b) Creates a new <div> element
    c) Deletes a <div> element
    d) Modifies a <div> element
  4. Which method removes an element from the DOM?
    a) removeChild()
    b) appendChild()
    c) querySelector()
    d) getElementsByTagName()
  5. What will document.querySelector("p").innerHTML = "Updated"; do?
    a) Delete all <p> elements
    b) Select all <p> elements and change their text
    c) Select the first <p> element and change its text
    d) Change the background color of <p> elements

Answers

  1. a) Document Object Model
  2. c) getElementById()
  3. b) Creates a new <div> element
  4. a) removeChild()
  5. c) Select the first <p> element and change its text

Assignment

Build a dynamic webpage using JavaScript to manipulate the DOM.

Instructions:

  • Create an HTML page with at least the following:
  • A heading (<h1>) that can change text when clicked.
  • A paragraph (<p>) that changes color on mouse hover.
  • A button (<button>) that adds a new list item to an unordered list (<ul>).
  • A button that removes the last item from the list.
  • Use JavaScript event listeners to handle interactions.
  • Submit the project as a ZIP file containing the HTML, CSS, and JavaScript file.

Assignment Answer

<!DOCTYPE html>
<html>
<head>
  <title>DOM Interaction</title>
  <style>
    p:hover { color: red; }
  </style>
</head>
<body>
  <h1 id="mainHeading">Click Me!</h1>
  <p class="description">Hover over this text.</p>
  <ul id="itemList">
    <li>Item 1</li>
  </ul>
  <button onclick="addItem()">Add Item</button>
  <button onclick="removeItem()">Remove Item</button>

  <script>
    document.getElementById("mainHeading").addEventListener("click", function() {
      this.innerText = "You clicked the heading!";
    });

    function addItem() {
      const ul = document.getElementById("itemList");
      const li = document.createElement("li");
      li.textContent = "New Item";
      ul.appendChild(li);
    }

    function removeItem() {
      const ul = document.getElementById("itemList");
      if (ul.lastChild) {
        ul.removeChild(ul.lastChild);
      }
    }
  </script>
</body>
</html>
