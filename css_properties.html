Quiz

  1. Which CSS property is used to change the text color of an element?
  2. How can you make a list without bullets?
  3. What property is used to change the background color of a div?
  4. Which CSS property controls the text size?
  5. How do you center a block element horizontally?

Answers

  1. color
  2. list-style-type: none;
  3. background-color
  4. font-size
  5. margin: 0 auto; (when width is set)

Assignment

Create a personal profile card using HTML and CSS. Your card should include:

  • A heading with your name
  • A short paragraph about yourself
  • A profile image
  • A list of 3 favorite hobbies

Style the card with:

  • Background color
  • Font styling
  • Rounded image
  • Shadow and padding

Assignment Sample Solution

<div class="profile-card">
  <h2>John Doe</h2>
  <img src="profile.jpg" alt="Profile Photo">
  <p>I am a web design enthusiast who loves coding and learning new things.</p>
  <ul>
    <li>Reading</li>
    <li>Drawing</li>
    <li>Cycling</li>
  </ul>
</div>
.profile-card {
  background-color: #f9f9f9;
  padding: 20px;
  width: 300px;
  box-shadow: 0 4px 8px rgba(0, 0, 0, 0.1);
  text-align: center;
  font-family: Arial, sans-serif;
}
.profile-card img {
  width: 100px;
  height: 100px;
  border-radius: 50%;
  margin-bottom: 10px;
}
