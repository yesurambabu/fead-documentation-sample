Special Characters

Introduction

JavaScript provides special characters (also called escape sequences) that allow developers to include special symbols, whitespace, and control characters within strings.

Commonly Used Special Characters in JavaScript

Special Character Meaning Example Output
\n New Line console.log("Hello\nWorld"); Hello
World
\t Tab Space console.log("Hello\tWorld"); Hello    World
\" Double Quote console.log("She said \"Hello\"."); She said "Hello".
\' Single Quote console.log('It\'s a sunny day.'); It's a sunny day.
\\ Backslash console.log("C:\\Program Files\\JavaScript"); C:\Program Files\JavaScript

Example: Using Special Characters in Strings

let message = "Hello,\nWelcome to JavaScript!";
console.log(message);

Output:

Hello,
Welcome to JavaScript!
let path = "C:\\Users\\Admin\\Documents";
console.log(path);

Output:

C:\Users\Admin\Documents

Why Are Special Characters Important?

  • They help format text properly.
  • They allow the inclusion of special symbols without breaking the code.
  • They improve readability and organization in strings.

Quiz: JavaScript Special Characters

  1. What special character is used to create a new line in a string?
    a. \t
    b. \\
    c. \n
    d. \r
  2. How can you include a double quote (") inside a string?
    a. Use '/"'
    b. Use \\
    c. Use \"
    d. Use '
  3. What does console.log("Hello\tWorld"); output?
    a. HelloWorld
    b. Hello    World
    c. Hello\nWorld
    d. Hello World (with no tab)
  4. Which escape sequence is used to insert a backslash (\) inside a string?
    a. \\
    b. \/
    c. \b
    d. \quote
  5. What will be the output of the following code?
    console.log("She said \"JavaScript is fun!\"");
    a. She said 'JavaScript is fun!'
    b. She said "JavaScript is fun!"
    c. Error
    d. JavaScript is fun!

Answers: JavaScript Special Characters Quiz

  1. c. \n
  2. c. \"
  3. b. Hello    World
  4. a. \\
  5. b. She said "JavaScript is fun!"
