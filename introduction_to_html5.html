Introduction to HTML5

What is HTML5?

HTML5 is the latest version of the HyperText Markup Language used to structure content on the web. It introduces new semantic elements, multimedia support, and browser-native APIs that make web development more powerful and accessible.

Features of HTML5

  • Semantic tags for meaningful markup
  • Support for audio, video, and canvas without plugins
  • Advanced form controls like date pickers and sliders
  • Offline storage and application cache
  • APIs for geolocation, drag-and-drop, and more

Basic HTML5 Document Structure

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>HTML5 Page</title>
</head>
<body>
  <h1>Hello HTML5!</h1>
</body>
</html>

This structure includes a doctype declaration, language attribute, UTF-8 encoding, and a simple body element.

New Semantic Elements

HTML5 introduced new elements that add meaning to the structure of your webpage:

Tag Purpose
<header> Introductory content for a section or page
<footer> Footer for a section or page
<article> Independent, self-contained content
<section> Generic grouping of related content
<nav> Navigation links
<aside> Related content, such as sidebars

Multimedia in HTML5

Audio

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  Your browser does not support the audio tag.
</audio>

Video

<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  Your browser does not support the video tag.
</video>

New Form Elements and Input Types

HTML5 adds input types that improve form usability and built-in validation.

Examples:

<form>
  Email: <input type="email"><br>
  Birthdate: <input type="date"><br>
  Favorite color: <input type="color">
</form>
  • email – Validates email address
  • date – Native date picker
  • color – Color picker interface

Graphics with <canvas>

The <canvas> element provides a drawing surface for JavaScript-powered graphics.

<canvas id="myCanvas" width="200" height="100"></canvas>

HTML5 APIs

HTML5 includes browser APIs that allow for richer web applications:

  • Geolocation API: Detects the user's location
  • Web Storage: Stores data locally in the browser
  • Drag and Drop: Enables native drag-and-drop functionality

Compatibility and Fallbacks

Not all browsers support every HTML5 feature. Use JavaScript libraries like Modernizr to detect features and implement fallbacks when necessary.
